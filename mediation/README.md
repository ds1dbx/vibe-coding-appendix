# 마음의 평안 - 명상 웹 앱

가볍게 명상을 할 수 있는 웹 페이지입니다.

## 기능

- 🌅 잔잔한 자연 이미지 슬라이드쇼 (10초마다 자동 전환)
- 🎵 배경 음악 (사용자 클릭 시 시작)
- ⏰ 3분 타이머
- ❤️ 좋아요 버튼 (마음에 드는 이미지에 하트 표시)
- 📝 명상 완료 후 명언 표시
- 🖥️ 전체 화면 몰입형 디자인

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 페이지를 클릭하여 배경 음악을 시작합니다.
3. 3분 동안 자연 이미지들을 감상하며 명상을 즐깁니다.
4. 마음에 드는 이미지가 나오면 ❤️ 버튼을 클릭하세요.
5. 3분이 지나면 아름다운 명언이 표시됩니다.

## 키보드 단축키

- `Space`: 좋아요 버튼 토글
