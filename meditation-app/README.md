# 마음의 평안 - 명상 웹 앱

가볍게 명상을 할 수 있는 웹 애플리케이션입니다.

## Cursor 프롬프트

```가볍게 명상을 할 수 있는 웹 페이지를 만들어 줘. 

- 잔잔한 음악이 배경으로 깔리면 좋겠어 
- 마음을 평안하게 해주는 이미지를 재생하면 좋겠어
- 각 이미지가 마음에 들면 "좋아요" 버튼을 누를 수 있게 버튼을 만들어 줘
- 3분 정도 명상을 하고 나면 명언을 하나 보여주고 음악을 종료해줘
```

## 기능

- ✨ 잔잔한 배경 음악 (음악 파일을 추가해야 합니다)
- 🖼️ 평안한 이미지 슬라이드쇼 (5초마다 자동 전환)
- ❤️ 각 이미지에 좋아요 버튼
- ⏰ 3분 타이머
- 💬 명상 완료 후 영감 주는 명언 표시

## 사용 방법

1. `index.html`을 브라우저에서 엽니다
2. "명상 시작하기" 버튼을 클릭합니다
3. 3분 동안 평안한 이미지와 음악을 즐깁니다
4. 마음에 드는 이미지가 나오면 좋아요 버튼을 누릅니다
5. 3분 후 자동으로 명언이 표시되고 음악이 종료됩니다

## 음악 파일 추가

현재 음악 파일이 없으므로 배경 음악 기능을 사용하려면:

1. `meditation-music.mp3` 파일을 프로젝트 폴더에 추가하세요
2. 저작권이 없는 잔잔한 음악을 사용하세요

대안으로 YouTube나 다른 음악 스트리밍 서비스를 사용할 수도 있습니다.
